# Screening Process — Founding Full-Stack Engineer

## Pipeline

```
Inbox → Screening (CEO, 20min) → Tech Challenge (asíncrono, 3-4h) → Técnica (CEO, 60min) → Referencias → Offer
```

## 1. Screening — 20 min

Preguntas guía:
- Cuéntame de un proyecto que hayas llevado de principio a fin
- ¿Cómo manejas prioridades con múltiples frentes?
- ¿Has trabajado en agencia o freelance?
- ¿Qué te llama la atención de Inadaptado?
- Rango salarial esperado
- Disponibilidad

**Pase:** Comunicación clara, experiencia real, actitud constructora, alineación salarial.

## 2. Technical Challenge — asíncrono (3-4h)

**Formato:** Repo privado GitHub, entrega vía PR.

**Challenge:** Landing page para un negocio local con:
- Next.js o Astro + TypeScript
- Tailwind CSS, responsive
- Formulario con Supabase
- Deploy en Vercel
- README con decisiones técnicas

**Bonus:** SEO, Lighthouse > 90, animaciones.

**Evaluación:**
- Código limpio (40%)
- Funcionalidad (30%)
- Decisiones técnicas (20%)
- Documentación (10%)

**Entrega:** 5 días.

## 3. Entrevista técnica — 60 min

1. Walkthrough del challenge (15min)
2. Arquitectura (20min): escalabilidad, seguridad, CI/CD
3. System design (15min): sistema de notificaciones para CRM
4. Q&A (10min)

## 4. Referencias

2 referencias laborales. Preguntar por: estilo de trabajo, fortaleza técnica, área de mejora, recomendación.

## 5. Offer

Verbal → offer letter → 5 días hábiles para firmar.

## Timeline objetivo

| Semana | Hito |
|---|---|
| 1 | Publicar oferta + difusión |
| 2-3 | Screening + Tech Challenges |
| 3-4 | Entrevistas técnicas |
| 4-5 | Referencias + Offer |
| 5-6 | Onboarding |
